# Google Sheets Setup Guide

This document explains how to set up the Google Spreadsheet that serves as the database for **मुंबई व उपनगर महानुभाव पंथीय संस्था**.

---

## 1. Create the Spreadsheet

1. Go to [sheets.google.com](https://sheets.google.com) and create a new blank spreadsheet.
2. Name it: `Mahanubhav Sanstha DB` (or any name you prefer).
3. You need **exactly three sheet tabs** with these names (case-sensitive):

| Tab name | Purpose |
|---|---|
| `Events` | Upcoming & past events |
| `Registrations` | Member/event registrations |
| `Gallery` | Photo gallery entries |

---

## 2. Events Sheet

**Tab name:** `Events`

Create these column headers in **Row 1** (columns A → J):

| Column | Header | Example value |
|---|---|---|
| A | `id` | `evt_001` |
| B | `title` | `चक्रधर स्वामी जयंती २०२६` |
| C | `titleEn` | `Chakradhar Swami Jayanti 2026` |
| D | `date` | `2026-08-15` |
| E | `time` | `09:00` |
| F | `location` | `मुंबई, दादर` |
| G | `description` | `वार्षिक महोत्सव...` |
| H | `imageUrl` | `https://...` (optional) |
| I | `isActive` | `TRUE` |
| J | `createdAt` | `2026-01-01T00:00:00Z` |

**Notes:**
- `id` is auto-generated by the Apps Script — leave blank when inserting manually.
- `isActive` must be `TRUE` or `FALSE` (Google Sheets checkbox or plain text).
- Date format: `YYYY-MM-DD`.

---

## 3. Registrations Sheet

**Tab name:** `Registrations`

Column headers in **Row 1** (columns A → L):

| Column | Header | Example value |
|---|---|---|
| A | `id` | `reg_001` |
| B | `name` | `राहुल शर्मा` |
| C | `email` | `rahul@example.com` |
| D | `phone` | `9876543210` |
| E | `address` | `दादर, मुंबई` |
| F | `mandal` | `दादर मंडळ` |
| G | `eventId` | `evt_001` |
| H | `eventName` | `चक्रधर स्वामी जयंती २०२६` |
| I | `members` | `4` |
| J | `notes` | `vegetarian food needed` |
| K | `status` | `pending` |
| L | `createdAt` | `2026-01-01T10:30:00Z` |

**Notes:**
- `status` values: `pending`, `confirmed`, `cancelled`.
- `members` = number of people in the group registration.
- `id` is auto-generated by the Apps Script.

---

## 4. Gallery Sheet

**Tab name:** `Gallery`

Column headers in **Row 1** (columns A → H):

| Column | Header | Example value |
|---|---|---|
| A | `id` | `gal_001` |
| B | `title` | `जयंती उत्सव २०२५` |
| C | `titleEn` | `Jayanti Festival 2025` |
| D | `imageUrl` | `https://drive.google.com/...` |
| E | `category` | `festival` |
| F | `eventId` | `evt_001` (optional) |
| G | `isActive` | `TRUE` |
| H | `createdAt` | `2025-08-15T00:00:00Z` |

**Notes:**
- `category` suggested values: `festival`, `mandal`, `service`, `other`.
- For `imageUrl`, use publicly shared Google Drive image links or any direct image URL.
- `id` is auto-generated by the Apps Script.

---

## 5. Verify the Setup

After creating all three tabs and headers:

1. Open **Extensions → Apps Script** in the spreadsheet.
2. Paste the contents of `backend/GOOGLE_APPS_SCRIPT.js`.
3. Change `ADMIN_TOKEN` at the top to a strong secret password.
4. Run the `setupSheets` function once (if present) to auto-populate sample data.
5. Deploy as a Web App (see `docs/deployment-guide.md`).

---

## 6. Sharing & Permissions

- The spreadsheet itself does **not** need to be public.
- Only the **Apps Script Web App** is public-facing.
- The Apps Script runs as **you (the owner)**, so it can read/write the private sheet.
- Never share the spreadsheet URL publicly — only the Apps Script URL goes in `frontend/src/services/api.ts`.
